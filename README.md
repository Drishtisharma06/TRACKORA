# Trackora 📍

### Real-Time Live Location Tracking & SOS Safety System

Trackora is a modern live location tracking and emergency response web application built using HTML, CSS, JavaScript, and Supabase.

It provides:

* 📍 Real-time location tracking
* 🚨 SOS emergency alerts
* 👤 User authentication & profile management
* 📊 Analytics dashboard
* 🔐 Secure password management
* 🛡️ User safety focused experience

---

# 🚀 Features

## ✅ Authentication

* User Signup/Login
* Secure Supabase Authentication
* Password Change System
* Persistent User Sessions

## ✅ Profile Management

* Editable user profile
